آموزش Blazor WebAssembly پیشرفته ( پروژه محور )

آموزش بلیزور (BLazor) پیشرفته پروژه محور

 

BLazor یکی از تکنولوژی های دیگر مایکروسافت است که با استفاده از آن میتوانید سایت های SPA پیاده سازی کنید.

مایکروسافت با استفاده از Blazor به شما این امکان را میدهد که به جای استفاده از فریمورک ها و کتابخانه هایی مثل React و ... با استفاده از کدهای سی شارپ Front End سایت خود را پیاده سازی کنید.

در این دوره به صورت پروژه محور Blazor رو یاد می گیریم.

 

برای کسب اطلاعات بیشتر لطفا ویدیو معرفی دوره را مشاهده فرمائید.
