آموزش مانیتورینگ و مدیریت لاگ ها در ASP.NET Core

هر برنامه وبی نیاز به نظارت دارد تا قبل از اینکه مشکلی کوچک تبدیل به یک مشکل بزرگ در سیستم شود بتوان جلوی آن را گرفت. شما باید بتوانید قبل از اینکه کاربران مشکلات سیستم را پیدا کنند، آن ها را یافته و رفع کنید.

در این دوره ابتدا اصول اولیه در رابطه با ورودی ها و اطلاعات مربوط به اکسپشن ها و HTTP Request را یاد می گیرید. سپس یاد می گیرید برنامه خود را آنالیز کنید، گزارش های کاربردی ایجاد کنید، از صفحات خطا مناسب استفاده کنید، مدیریت API Error ها را یاد می گیرید، از لاگ ها و پلتفرم های سودمند برای نظارت برنامه خود استفاده خواهیم کرد. همچنین پیاده سازی و تنظیم مانیتورینگ در برنامه یتان را نیز به شما آموزش خواهیم داد. شما مهارت و دانش لازم برای ثبت وقایع و نظارت برنامه را کسب می کنید تا بتوانید همیشه و به خوبی برنامه های خود را پشتیبانی کنید.

پیشنهاد می کنیم ابتدا ویدیوی معرفی دوره را مشاهده نمایید.
