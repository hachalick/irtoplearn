آموزش Microsoft Azure DevOpsبه همراه پروژه عملی

دوره آموزش آژور دواپس(Azure DevOps) از 0 تا 100

 

خب دوستان دواپس  مجموعه ای از روش ها و ابزار هایی هست که همکاری و ارتباط رو برای ما آسان کرده و بهمون کمک میکنه پروژه خودمون رو در کمترین زمان با بالاترین کیفیت آماده بکنیم.

خب از قابلیت هایی که  Azure DevOps در اختیار ما قرار میده

Azure Pipelines

Azure Boards

Azure Repos

Azure Artifacts

که ما توی این آموزش کار با تمامی این قابلیت ها رو به شما یاد میدیم.

 

در این دوره چه مواردی رو آموزش میدیم؟

 

1-در این دوره هم با Azure Devops Service کار می کنیم هم با Azure Devops Server تا کار با هر دو اینا رو یاد بگیرید .

2- در این دوره با سرویس های مختلفی کار می کنیم مثل Azure Repos و Azure Pipeline و Azure Board و... که شما کار با این سرویس ها رو به صورت کامل یاد می گیرید . 

3-در این دوره Ci و Cd به صورت کامل آموزش داده شده اینکه چطور ساختار Ci و Cd رو برای پروژه های خودمون ایجاد بکنیم .

4- در این  دوره موارد امنیتی مربوط به بخش های مختلف بررسی شده و شما بعد از این دوره به راحتی می تونید تیم های خودتون رو مدیریت و کنترل بکنید .

5- در آخر این دوره دو تا پروژه عملی داره تا شما بتونید کار با Azure Devops رو به خوبی یاد بگیرید و بتونید ازش استفاده بکنید .

  

پیشنهاد میکنیم جهت آشنایی بیشتر با دوره بخش اول ( معرفی دوره ) مشاهده کنید .
